# Product Overview — XOGame

## What it is
A simple, browser-based 5×5 tic tac toe game. Two players take turns marking X and O on a 5×5 grid. First to get five in a row wins. No accounts, no installs, no friction.

## Users / Audience
Anyone who wants a quick casual game — kids, friends, coworkers killing time.

## Positioning
Minimal and instant. Opens in a browser, works immediately, looks clean. No ads, no sign-up, no bloat.

## Core Feature
- 3×3 interactive game board
- Turn-based two-player gameplay (X vs O, same device)
- Win detection (row, column, diagonal) and draw detection
- Draw detection when board is full
- Game reset / play again button
- Current player indicator
- Session scoreboard (X wins / draws / O wins) — toggleable via settings

## Brand & Tone
Casual, playful, clean. Simple color palette. No unnecessary decoration.

## Scope
Single-page app. No backend, no auth, no persistence. Pure frontend game logic.

## Strategic Principles
- Zero friction: load and play immediately
- Simple wins: nothing that isn't needed for the core game loop
