Core Features

Everything the game needs.
Nothing it doesn't.

🎮 Casual Players 👨‍👩‍👧 Friends & Family 🏢 Coworkers
🟦

5×5 Game Board

Interactive 5×5 grid — tap any empty cell to place your mark. First to get five in a row wins.

🔄

Turn Indicator

Always shows whose turn it is — X or O — so there's never confusion mid-game.

🏆

Win Detection

Automatically detects wins across all rows, columns, and both diagonals. Highlights the winning line.

🤝

Draw Detection

Recognizes a full board with no winner and calls it a draw — no dead states.

↩️

Play Again

One-tap reset clears the board and starts a fresh game without reloading the page.

📊

Session Scoreboard

Tracks X wins, draws, and O wins across the session. Toggleable — keep it or hide it.
